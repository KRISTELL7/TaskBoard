# TaskBoard
Es una aplicacion para gestionar tareas, en la cual se puede agregar,editar y eliminar una tarea cuando  ya este marcada como completada

#Instalación
1. Clonar el repositorio de git https://github.com/KRISTELL7/TaskBoard.git
2. entrar a la carpeta 
3. instalar dependencias ( provider,shared_preferences,uuid)
4. ejecutar con un emulador o dispositivo de prueba.

#Emulador
Para las pruebas utilice Android Studio(pixel 4), asi mismo para evitar errores en las vistas utilice un dispositivo fisico(infinix 60).
